A proposta do projeto é:

Desenvolva operações CRUD para instanciar objetos peças em um SGBD relacional. Há peças genuínas e peças de segunda linha, todos os tipos de peças possuem código, nome e preço. 
Peças genuínas são de origem Brasileira, já as de segunda linha são de outros países.
O preço de peças brasileiras são 5% maior que o preço das peças estrangeiras. 
Para esse desenvolvimento explore Interface e tratamento de exceção, além da implementação das classes utilizando padrões POO em Java e a classe Scanner
